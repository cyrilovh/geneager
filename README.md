### Architecture MVC
public --> all public ressources (Javascript, CSS, Images, ...)

src --> All my programs (view / model / controller)

view: HTML (with few php)

model: class

controller: logic program

### Requirement
Apache2:
- RewriteEngine required

PHP mods: 
- Imagick recommened
- Curl recommended
