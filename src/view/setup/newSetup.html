Installation de Geneager CMS !

Le site n'est pas encore configuré.

Étape 1 — Base de données

Pour commencer importer la base de données MySQL via PHPMyAdmin par exemple

Étape 2 — Fichier de configuration

Renommez le fichier « config.sample.php » en « config.php » et éditez les informations.

AVERTISSEMENT:

Pensez à modifier les valeurs des superglobales SALT_PASSWORD et PASSWORD_TOKEN dans le fichier de configuration.

Il s'agit d'un mot de passe pour chiffrer (« crypter ») les données envoyées ou stockées dans la base de données.

Mettez une valeur complexe impossible à deviner.
Exemple: vBspPq5*q6zzPma!-s9

Évitez les mots de passe contitués uniquement de chiffres et de lettres.
Il n'est pas nécessaire de le mémoriser.

INFORMATIONS COMPLEMENTAIRES:

Configurez la base de données en éditant les variables suivantes dans le fichier de configuration:

  • $db_name: nom de la base de données)
  • $db_host: adresse IP ou nom d'hôte de la base de données (généralement « 127.0.0.1 » ou « localhost »)
  • $db_user: nom d'utilisateur de la base de données
  • $db_password: mot de passe

INFORMATION COMPLEMENTAIRE:

Pensez à vérifier que la superglobale PROD dans le fichiers de configuration soit sur TRUE si ne vous faites pas du développement ou déboguage.

La personnalisation des autres éléments est configurable via la page « paramètres » dans la partie administration du site (via un compte admin).
